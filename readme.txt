Project:
This project analyzes Starbucks marketing data and provides a recommendation
for a customer targeting strategy to improve performance of promotions. The
project is based on simulated customer data and transactions of these
customers with the app.

Files:
4 python notebooks numbered from 1 to 4
Project Report.pdf and Proposal.pdf - informational overview
Various data files in the subfolders

Data:
Each python notebooks' results are saved in a different data folder.

Environment:
These notebooks were ran in the AWS Sagemaker module. The PCA and K-means
implementation require being run in Sagemaker. The free tier was used when
developing these notebooks and should be sufficient for any user who wishes
to run them.
